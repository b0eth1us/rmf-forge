# ASD STIG Data

Place the Application Security and Development STIG XCCDF benchmark here as:
  `ASD_STIG.xml`

Download from: https://public.cyber.mil/stigs/downloads/
Search: "Application Security and Development"

The file is not bundled in this repo because DISA STIGs are subject to their
distribution terms. Add it locally before building the Docker image, or mount
it as a volume in development.
